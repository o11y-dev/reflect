Today

Hosted reports, observation scores, and skill/agent generation driven by telemetry. Reads OTLP and local session stores; runs in your terminal or browser.

opentelemetry-hooks

The OpenTelemetry instrumentation layer that captures AI coding agent activity and emits OTLP. The data foundation underneath reflect — bring any OTLP-compatible backend.

Install

Skills are one distribution channel, not the identity. Install as a Claude skill, via pipx, or from source — all three produce the same reflect.

reflect (reports & scores)

pipx install o11y-reflect
reflect setup
reflect

# no telemetry yet?
reflect --demo

opentelemetry-hooks (capture only)

pipx install opentelemetry-hooks
opentelemetry-hooks install
